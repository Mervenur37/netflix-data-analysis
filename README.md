# Netflix İçerik Analizi

Netflix veri seti üzerinde veri temizleme, görselleştirme ve makine öğrenmesi uygulaması.

## Proje İçeriği
- 8807 Netflix içeriğinin analizi
- Eksik veri temizleme
- Film vs dizi dağılımı
- Yıllara göre içerik büyümesi
- En çok içerik üreten ülkeler
- En popüler türler
- Random Forest ile içerik tipi tahmini (%73.6 doğruluk)

## Kullanılan Teknolojiler
Python, Pandas, Matplotlib, Seaborn, Scikit-learn

## Veri Seti
[Netflix Movies and TV Shows - Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)
